Promises

You print #PunkMoney by tweeting it into existence. For example:

@someone I promise you one cold beer. Expires in 1 month. #punkmoney

#PunkMoney will appear in the ticker no more than a minute later.

The easiest way to create #punkmoney is to use the printer. This guarantees the ticker will find it, and is faster than typing it out. If you prefer to do it yourself, the rules are simple enough:

  • Your tweet must begin with the Twitter name of the recipient (e.g. @someone)
  • It must contain the word 'promise'
  • It must contain the hashtag #punkmoney

    • These parts can be in any order.

    Optional:
  • You can add an expiry in the format 'Expires in [number] [minutes/hours/days/months/years]'
  • To make a note non-transferable add 'NT' before the hasthag.

If the ticker doesn't find yours within a minute, there was probably a syntax error in it.

Transfers

If you've received some #PunkMoney and want to transfer it to someone else, just reply to the original note, or to any existing replies to it, with:

@someone transfer @recipient #punkmoney

The rules are:

  • Your tweet must be a reply to the original note, or to another reply to it
  • It must contain the word transfer
  • It must contain the username of the recipient, (e.g. @recipient)
  • It must end with the hashtag #punkmoney

Transfers are only allowed to trusted users. (Anyone can create and redeem #punkmoney, but transfers are currently limited to existing members.) If a note has expired, has already been redeemed or is marked as non-transferable, the transfer won't be acnkowledged by the tracker.

Thanks

To thank someone after redeeming #PunkMoney, reply to the original note, or to another reply to it, with

@issuer thanks #punkmoney

The rules:

  • Your tweet must be a reply to the original note, or to another reply to it
  • It must contain the word thanks
  • It must contain the hashtag #punkmoney

If the note is open, and you're its current bearer, the ticker will note the redemption within a minute.

Alternatively, you can thank by favouriting the original tweet. This usually takes the tracker about 45 minutes to acknowledge.

To issue a thank you statement which isn't connected to a promise, simply tweet

@someone thanks for the chocolate #punkmoney

Offers/Needs

To create an offer, tweet

offer guitar lessons #punkmoney

To create a need, tweet

need help mowing the lawn #punkmoney

The rules:

  • Your tweet must contain the word 'offer' (or 'need')
  • It must contain something you are offering (or needed)
  • It must contain the hashtag #punkmoney

To close an offer/need (thus removing it from your profile), reply to the orignal tweet with

close offer/need #punkmoney

Record trust

#PunkMoney maintains a TrustNet, which gets updated automatically. The ticker will only acknowledge transfers to users who are in it.

The TrustNet is managed collectively by the community. It's used to keep #PunkMoney circulating within a circle of trust, so that issuers know their notes will stay within the community.

To be included, an existing member needs to add you to their own punkmoney-trusted Twitter list. A maximum of 20 users can be added by a single referrer. The tracker will find and add you within 30 minutes. You will then be able to add new users yourself.
